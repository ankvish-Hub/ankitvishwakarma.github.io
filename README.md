🚀 Overview  
This is my **personal portfolio website** built using **React (Vite)** and **Tailwind CSS**.  
It highlights my **skills, experience, certifications, and projects**, designed with **smooth animations** and a **custom cursor effect** for an interactive experience.  



🛠️ Tech Stack 

| Category | Technology |
|-----------|-------------|
| **Framework** | React 19 + Vite |
| **Styling** | Tailwind CSS |
| **Animations** | Framer Motion, Custom Cursor Animation |
| **Icons** | React Icons |
| **Typing Effects** | React Type Animation |
| **Linting** | ESLint |

 📁 Sections  

| Section | Description |
|----------|--------------|
| 🏠 **Navbar** | Sticky navigation bar with smooth scroll |
| 🙋‍♂️ **About** | Short developer intro |
| 💼 **Experience**| Work & internship timeline |
| 🧠 **Skills**| Tech stack with icons |
| 🧾 **Certifications** | Certificates with preview cards |
| 🚀 **Projects** | Featured projects with live demo links |
| ✉️ **Contact** | Email & social media links |
| ⚙️ **Footer** | Copyright & credits |



✨ Features  

✅ Responsive Design (Mobile-Friendly)  
✅ Smooth Page Transitions with Framer Motion  
✅ Interactive Cursor Animation (`utils/cursorAnimations.jsx`)  
✅ Modern Typing Effect  
✅ Fast Build with Vite  
✅ Modular and Clean Code Structure  



 ⚙️ Installation & Setup  

```bash
# Clone the repository
git clone https://github.com/ankvish-Hub/portfolio_dev

# Move into the project directory
cd portfolio_dev

# Install dependencies
npm install

# Start the development server
npm run dev

Then open 👉 http://localhost:5173/ in your browser.
